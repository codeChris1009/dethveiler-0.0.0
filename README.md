## 專案簡介

- 以 React + TypeScript 開發的現代化天氣地圖應用
- 整合 OpenWeather API、Mapbox、ShadCN UI
- 支援主題切換、單位系統、地圖互動、即時天氣圖表
- 高度模組化，結構清晰，易於擴展與維護

---

## 核心功能

- 小時天氣分頁（溫度、體感、降水、風、濕度、雲量、壓力、UV、能見度）
- 主題色與 className 動態覆蓋，設計一致
- Mapbox 地圖定位、標記、卡片元件
- 完整 hooks、context 狀態管理
- API 層分離，統一請求與錯誤處理
- 全功能配置與單位系統管理
